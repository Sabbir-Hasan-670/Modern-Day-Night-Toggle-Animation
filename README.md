# 🌓 Modern Day/Night Toggle Showcase

Welcome to the **Day/Night Toggle Animation Showcase**! This repository features two distinct versions of a smooth, interactive day and night switcher component, designed using cutting-edge frontend methodologies.

---

## 🚀 Projects Overview

| Version | Design Style | Highlights |
| :--- | :--- | :--- |
| **Version 1** | Dynamic Custom SVG / Minimalist UI | Smooth morphing transitions, lightweight footprint, and crisp vector rendering. |
| **Version 2** | Glassmorphism & Cyberpunk Neon Vibe | Premium glow effect shadows, custom CSS properties, and immersive transition layers. |

## 🛠️ Tech Stack & Architecture

- **Markup:** HTML5 (Semantic Structure)
- **Styling:** CSS3 (Custom Variables, Flexbox, Keyframe Animations, Pseudo-elements)
- **Interactivity:** JavaScript ES6+ (Event Handlers, Document Object Model (DOM) State Control)
